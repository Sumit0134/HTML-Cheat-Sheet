What is HTML?

  • HTML stands for HyperText Markup Language.
  • It is used for creating webpages.
  • It is a markup language with no programming syntax. The language provides a structure to design webpages.
  • Using HTML, one can define elements such as headings, paragraphs, links, images, etc.

Example of HTML:-

<!DOCTYPE html>

<html>

<head>

<title>HTML CHEAT SHEET</title>

</head>

<body>

<h1>Example of HTML</h1>

<p>This is an example of HTML.<p>

</body>

</html>

Output on Browser:-

HTML CHEAT SHEET

Example of HTML

This is an example of HTML.

Classification of above program:-

  • <!DOCTYPE html> - it tells that this is an HTML document.
  • <html> - it is the starting element of an HTML document under which all other elements are present.
  • <head> - it contains information about the page such as title of the page, metadata, etc.
  • <title> - it contains the title of the page which is shown in the browser's title bar.
  • <body> - it contains the main content of the page.
  • <h1> - it is a heading tag used for defining heading of a page.
  • <p> - it is a paragraph tag used for defining paragraph of a page.

Features of HTML:-

  • It helps to organize the contents of a webpage using various elements such as headings, paragraphs, lists, tables, etc.
  • It supports multimedia such as text, images, audio, video, etc.
  • It is flexible which means it can be used with other technologies such as CSS, JavaScript, etc.
  • It is compatible which means it can run in any browser.

How to create an HTML file?

To create an HTML document, <fileName>.html extension is used. For example, index.html.

What are HTML elements?

The opening tag, the closing tag, the content, and the attributes are called HTML elements.
For example,
                      <p>This is a content.</p>

Output on Browser:-

This is a content.

Here,
  • <p> is an opening tag.
  • "This is a content." is a content.
  • </p> is a closing tag.
HTML elements are hierarchical which means they can be nested inside one another to create a tree-like structure of the contents on the webpage.

  1. Paragraph Tag

    2. The HTML <p> tag is used to create paragraphs.

    The headings <h1> to <h6> tags are block-level elements. They start on a new line.
